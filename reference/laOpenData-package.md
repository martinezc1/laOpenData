# laOpenData: Convenient Access to Los Angeles Open Data API Endpoints

Provides simple, reproducible access to datasets from the Los Angeles
Open Data portal <https://data.lacity.org/>. Functions return results as
tidy tibbles and support optional filtering, sorting, and row limits via
the Socrata API.
